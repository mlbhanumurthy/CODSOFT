# Chatbot with Rule-Based Responses
## Overview
A simple chatbot that responds to user messages using predefined rules and pattern matching. The chatbot identifies keywords in the user's input and generates appropriate responses without using machine learning models.

## Features
* Responds to greetings and common questions
* Provides predefined answers
* Displays real-time conversation
* User-friendly interface
* Beginner-friendly implementation

## Technologies Used
* HTML
* CSS
* JavaScript

## How It Works
1. User enters a message.
2. The chatbot checks the input for specific keywords.
3. A matching response is selected using if-else conditions.
4. The response is displayed in the chat window.

## Learning Outcomes
* Basic Natural Language Processing concepts
* Pattern matching techniques
* JavaScript event handling
* Conversation flow design


# Tic-Tac-Toe AI
## Overview
An AI-powered Tic-Tac-Toe game where a human player competes against an unbeatable computer opponent. The AI uses the Minimax algorithm to evaluate all possible moves and select the optimal one.

## Features
* Human vs AI gameplay
* Unbeatable AI opponent
* Automatic win, loss, and draw detection
* Restart game functionality
* Interactive game board

## Technologies Used
* HTML
* CSS
* JavaScript

## How It Works
1. The player makes a move as X.
2. The AI evaluates all possible future game states using the Minimax algorithm.
3. The best move is selected and played as O.
4. The game continues until a win or draw occurs.

## Learning Outcomes
* Game theory fundamentals
* Minimax search algorithm
* Decision-making in AI
* Recursive programming concepts
* Interactive web development

## Future Enhancements
* Alpha-Beta Pruning for optimization
* Difficulty levels
* Score tracking system
* Multiplayer mode
* Enhanced UI and animations
